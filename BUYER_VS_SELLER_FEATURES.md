## ğŸ”‘ KEY DIFFERENCES

### 1. **Primary Goal**
- **Buyer**: Find and purchase a business
- **Seller**: Sell their business

### 2. **Verification Process**
- **Buyer**: Optional, instant access
- **Seller**: Mandatory KYC, requires admin approval

### 3. **Subscription Limits**
- **Buyer**: Connection limit (10/25/unlimited per month)
- **Seller**: Listing limit (5/10/unlimited total)

### 4. **Connection Flow**
- **Buyer**: Initiates requests, waits for approval
- **Seller**: Receives requests, approves/rejects

### 5. **Data Access**
- **Buyer**: Limited until connection approved
- **Seller**: Full access to own listings always

### 6. **Analytics**
- **Buyer**: No analytics
- **Seller**: Detailed listing and profile analytics

### 7. **Content Creation**
- **Buyer**: Cannot create listings
- **Seller**: Can create and manage listings

### 8. **Approval Process**
- **Buyer**: No approval needed (except email verification)
- **Seller**: Requires admin approval for:
  - Account verification
  - Each listing
  - Listing edits

## ğŸ’¡ Important Notes

### For Buyers
1. **Subscription is mandatory** to send connection requests
2. **Connection limit** resets monthly
3. **Full listing details** only visible after seller approval
4. **Cannot contact sellers** directly without connection
5. **Can save unlimited listings** regardless of plan

### For Sellers
1. **KYC verification required** before creating listings
2. **All listings need admin approval** before publishing
3. **Listing edits also need approval** (creates pending edit)
4. **Can view own analytics** in real-time
5. **Can initiate connections** to interested buyers
6. **Listing limit** based on subscription tier

### For Both
1. **Can block each other** to prevent communication
2. **Blocked users** cannot see listings or send messages
3. **Subscription required** for core features
4. **Email verification required** for all accounts
5. **Can manage notification preferences**
